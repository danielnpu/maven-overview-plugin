overview:help

Full name:

com.agilejava.maven.plugins:maven-overview-plugin:1.1-SNAPSHOT:help

Description:

Display help information on 'com.agilejava.maven.plugins:maven-overview-plugin' plugin. Call 'mvn overview:help -Ddetail=true' to display all details.

Attributes:

  • Requires a Maven 2.0 project to be executed.

Optional Parameters

Name Type Since Description
detail boolean - If true, display all settable properies for each goal. Default value is: false.

Parameter Details

detail :

If true, display all settable properies for each goal.

  • Type: boolean
  • Required: No
  • Expression: ${detail}
  • Default: false
