overview:overview

Maven Overview Plugin.

Generates overview graph of project dependencies.

Mojo Attributes:

  • Requires a Maven 2.0 project to execute.
  • Automatically executes within the lifecycle phase: site

Optional Parameters

Name Type Description
artifactCollector ArtifactCollector Artifact collector, needed to resolve dependencies.
artifactFactory ArtifactFactory No description.
artifactMetadataSource ArtifactMetadataSource No description.
dependencyTreeBuilder DependencyTreeBuilder No description.
excludes String Artifacts to be excluded. Regular expressions matching the full artifact designation.
height int Rendered graph height in pixels. Default value is 1200.
localRepository ArtifactRepository Local Maven repository.
project MavenProject Maven Project. Default value: ${project}
siteRenderer Renderer Generates the site report.
suppressedScopes String Suppressed scopes. Scopes that are not supposed to be shown on graph as edge labels.

vertexFullLabel boolean Should vertex name be full artifact ID. Default value is false.
width int Rendered graph width in pixels. Default value is 1200.
