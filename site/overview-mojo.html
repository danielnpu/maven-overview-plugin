overview:overview

Note: This Mojo should be used as a Maven report.

Full name:

com.agilejava.maven.plugins:maven-overview-plugin:1.1-SNAPSHOT:overview

Description:

Maven Overview Plugin.

Generates overview graph of project dependencies.

Attributes:

  • Requires a Maven 2.0 project to be executed.
  • Binds by default to the lifecycle phase: site.

Optional Parameters

Name Type Since Description
excludes String - Artifacts to be excluded. Regular expressions matching the full artifact designation.
height int - Rendered graph height in pixels. Default value is: 1200.
localRepository ArtifactRepository - Local Maven repository.
project MavenProject - Maven Project. Default value: ${project}
suppressedScopes String - Suppressed scopes.

Scopes that are not supposed to be shown on graph as edge labels.

vertexFullLabel boolean - Should vertex name be full artifact ID. Default value is: false.
width int - Rendered graph width in pixels. Default value is: 1200.

Parameter Details

excludes :

Artifacts to be excluded. Regular expressions matching the full artifact designation.

  • Type: java.lang.String
  • Required: No
  • Expression: ${excludes}

height :

Rendered graph height in pixels.

  • Type: int
  • Required: No
  • Expression: ${height}
  • Default: 1200

localRepository :

Local Maven repository.

  • Type: org.apache.maven.artifact.repository.ArtifactRepository
  • Required: No
  • Expression: ${localRepository}

project :

Maven Project. Default value: ${project}

  • Type: org.apache.maven.project.MavenProject
  • Required: No
  • Expression: ${project}

suppressedScopes :

Suppressed scopes.

Scopes that are not supposed to be shown on graph as edge labels.

  • Type: java.lang.String
  • Required: No
  • Expression: ${suppressedScopes}

vertexFullLabel :

Should vertex name be full artifact ID.

  • Type: boolean
  • Required: No
  • Expression: ${vertexFullLabel}
  • Default: false

width :

Rendered graph width in pixels.

  • Type: int
  • Required: No
  • Expression: ${width}
  • Default: 1200
